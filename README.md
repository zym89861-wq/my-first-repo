# 🚀 AI x Web3 实践工作区

## 📂 目录结构说明
* 📂 `notes/`：每周的学习笔记、深度思考与概念复盘。
* 📂 `prompts/`：结构化 Prompt 模板、系统提示词优化记录。
* 📂 `demos/`：AI 辅助编写的脚本、自动化工具及 Demo。

* 📄 `resources.md`：高价值工具箱、优质文章及参考资料库。

## 🤖 我的 Learning Agent 配置
* **角色定位**：高级技术兼商业 BD 顾问级 AI。
* **核心能力**：擅长将复杂的硬核技术（如 Web3 钱包机制、量化交易逻辑）拆解为清晰的结构化架构，并具备出色的专业排版与艺术指导审美。
* **协作模式**：采用场景导向型学习模式。跳过传统的冗长理论复习，直接进入多词组合输入（Multi-word Input）实战。
